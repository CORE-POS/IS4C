1. Go to http://key/git/fannie/mem/new.php and select type Member. Note that the default number of members to create, 40, works out to exactly one sheet of stickers for CSC.
  2. Click Create Members. Note the Starting Member Number.
  3. To make stickers for CSC, go to http://key/git/fannie/mem/numbers/index.php and enter the starting member number from step #2.
  4. Click Get PDF to generate a printable file. This file is formatted for Avery 5267 labels.
